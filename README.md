# Glasshouse // Infrastructure & Engineering

Static landing page for Glasshouse Concepts.

## Local Development

```bash
npm install
npm run dev
```

Opens at `http://localhost:3000`.

## Deploy to Vercel

1. Push this repository to GitHub.
2. Go to [vercel.com](https://vercel.com) and click **Add New Project**.
3. Import the GitHub repository.
4. Vercel auto-detects Vite — click **Deploy**. No extra settings needed.

## Making Changes

Edit `index.html` for content and structure.  
Edit `src/style.css` for custom styles and animations.  
Tailwind utility classes are available directly in the HTML.
